Rendering

File Formats

Experio offers support for .obj, .bmp, .tga, and .fnt files, with .png, .jpg, .svg, .fbx, .vox, and .ttf support on the way.

Basic Rendering

Experio offers support for a variety of rendering components, including meshes, billboards, and particles

Camera System

Experio offers a camera system that natively supports camera switching and interpolation between cameras

Material System

Experio offers a materials system that allows users to create and define their own materials.

Post Processing

Experio offers support for a basic post-processing pipeline
